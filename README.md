## 概要

prisma-zod-mockは、Prismaスキーマから型安全なモックデータを生成するためのPrismaジェネレーターです。

### 主な機能

- 🎯 **セマンティック推論**: フィールド名から自動的に適切なデータ型を推論
- 🔧 **カスタマイズ可能**: `@mock`アノテーションで細かい制御が可能
- 📦 **複数出力形式**: 単一ファイル/複数ファイル出力をサポート
- 🇯🇵 **日本語対応**: Faker.jsの日本語ロケールを使用
